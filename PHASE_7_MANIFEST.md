## Red Flags - Stop Trading If

- [ ] Margin level < 500%
- [ ] Daily loss > 5%
- [ ] Drawdown > 10%
- [ ] EA crashes or errors
- [ ] Connection lost > 5 min
- [ ] Win rate < 30%
- [ ] Consecutive losses > 5
- [ ] Largest loss > 3%
- [ ] Slippage > 5 pips
- [ ] Trades not logging

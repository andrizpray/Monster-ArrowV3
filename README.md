# MonsterArrows V3 - Expert Advisor

Automated trading Expert Advisor for MetaTrader 5 based on Smart Money Concepts (SMC).

## Quick Start

1. **Compile** `MonsterArrows_V3_EA.mq5` in MetaEditor
2. **Attach** to chart on demo account
3. **Monitor** for 1+ weeks
4. **Deploy** to live account if GO

- **Order Management**: Auto entry, SL/TP, position sizing
- **Risk Control**: Daily loss limit, drawdown, trade expiry
